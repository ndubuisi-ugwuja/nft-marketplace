This project is a **Next.js-powered NFT Marketplace** that allows users to mint, list, buy, and manage NFTs using modern Web3 tools such as **Wagmi**, **Viem**, **Alchemy**, and **Smart Contracts** deployed on test networks.

This repository is currently **under active development**, and breaking changes may occur frequently.
